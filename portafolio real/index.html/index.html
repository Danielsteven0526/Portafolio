Introducción

Nuestra aplicación facilitará a un grupo de usuarios interesados en el fútbol en Bogotá encuentros para partidos de fútbol 5, 11 y microfútbol. Los usuarios podrán crear equipos, registrar sus datos y escaneo facial para mayor seguridad. La app usará GPS para encontrar partidos en zonas cercanas, además tendrá chat, apuestas con depósito seguro y calificación post partido.

Propósito y Justificación

Uno de los problemas más recurrentes para jugar partidos de fútbol es la dificultad para concretarlos. Queremos generar una aplicación que permita coordinar partidos y personas de manera organizada e intuitiva.

Objetivos

General

Facilitar encuentros para partidos de fútbol a través de una aplicación móvil que use GPS para buscar personas con el mismo interés.

Específicos

  • Generar una aplicación usando Python.
  • Desarrollar una base de datos sólida para un registro seguro.
  • Implementar reconocimiento facial para mayor confiabilidad.
  • Generar estadísticas del proyecto.
  • Usar GPS para buscar en zonas cercanas.
  • Promover el deporte y la interacción social.

Pregunta Problema

¿Te ha pasado que quieres jugar un partido de fútbol, pero todas las canchas están llenas o no hay gente?

Alcance

Desarrollar una aplicación que permita organizar encuentros y partidos de fútbol, automatizando gestión de fecha, hora y lugar, con una interfaz didáctica. Además, se planea incluir diseños para equipos y ampliar zonas de búsqueda.

Marco Teórico

La aplicación usará localización para generar encuentros deportivos, con apuestas gestionadas por una API. Se usará Python, GPS y reconocimiento facial para seguridad y confianza. Esto promoverá el deporte y las relaciones entre personas.

Caracterización - Análisis de Datos Encuesta

Se realizó una encuesta para conocer el interés y preferencias de los usuarios potenciales. Los datos indican un público mayormente joven (18-24 años) y masculino, con alta aceptación para el uso de registro seguro y la aplicación en general.

Conclusión Análisis de Datos

La encuesta confirma la necesidad de un sistema de registro seguro con escaneo facial y la viabilidad de la aplicación, con un 70% de aceptación. Se recomienda que solo personas mayores de 18 años usen la app, especialmente por las apuestas.
